other interesting property

if

n : List x -> Maybe x f : x -> y

then

(n >>> map f) == (map f >>> n)

in english

if

n is a function between collections

f is a function from x to y

then

it does not matter when we transform the elements of the collection with f, before or after n, the result is the same

such properties may be used for example as justification of program transformations to improve runtime efficiency

this particular property of parametric functions between collections can be proved [with mathematical rigour] in a much more general form, from arbitrary collection to arbitrary collection
