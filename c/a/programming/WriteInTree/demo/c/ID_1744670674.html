Create a mindmap file with some arbitrary content, just a simple node like "Hello World" suffices.

The Language Node

The first child of the root node must be the language node: "# language". It must have exactly 2 children: "write-in-tree" and "x.y" where x, y are integers denoting the version of the language used in the file.

Compile

Call this compiler with the command line: "write-in-tree.exe --web -i <path-to-the-saved-mindmap-file> -o <path-of-the-output-folder>"

Advice: put this command line into a script file for saving and possible later extensions. This script file in this tutorial wil be referred to as the "compile script".
