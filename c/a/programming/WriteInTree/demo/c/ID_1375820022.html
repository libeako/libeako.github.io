Demonstration and Tutorial of WriteInTree → Outline Editors →

The input data format of the compiler is a tree of text fragments ["text nodes"]. This data format is generally named as "outline". There are many outline editors, but 2 of them sticks out from the crowd. This compiler currently support the data format of only these 2.

They are:

Freeplane

Freemind

They are both more than just outline editors, they are mindmap editors. Still they can be used simply as outline editors. They both are free software, they are mature, though they do contain bugs. They run on Java platform. If you do not have your own preference among these 2, I recommend Freeplane.
