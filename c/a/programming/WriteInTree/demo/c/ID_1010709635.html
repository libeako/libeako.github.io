Text

They are simply just entered without any special formatting.

Image

Represented by an "# image" node. This node must have exactly 1 chlid, a plain text node containing the path to the image file.

Links

Any inline element may be a link. To make a node link, attach exactly 1 child to it, a "# links-to" node. This node should have exactly 2 children:

The type of link: "external" or "internal".

A plain text node containing the address where the link to point to.

Internal Address

An internal address is a user-defined unique name of a node. To give a name to a node, attach an "# id" child node to it having exactly 1 child: a plain text node containing the given name.
