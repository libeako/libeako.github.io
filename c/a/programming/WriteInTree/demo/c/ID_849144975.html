The links to the generated html pages and to their particular elements are stable, which means that normal editing of your work will not invalidate the links to parts of the generated html website.

Explanation

The compiler reads from the mindmap file not only the text content of the nodes, but also the unique identifiers of them too, which are generated by the mindmap editor. While the mindmap is edited, the identifiers of the nodes stay the same. And the links are created by the compiler using only these node identifiers.
