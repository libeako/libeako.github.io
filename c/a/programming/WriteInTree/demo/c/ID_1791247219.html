This is a software that transforms mindmap into html website. The name "write-in-tree" is meant to mean both a technology and a concrete software [the "compiler"].

This software [the whole thing] is free.

Is for those who want to write a book [or article], want to write and maintain it fast, without hassle, but beside the comfort of author also want to give the readers a convenient html website format.

This tutorial itself is WrittenInTree.

Though the compiler is not mature yet, the author of it has already used it to write his book without any major problems and with great satisfaction.

The current version of the language described in this tutor is 1.1.
