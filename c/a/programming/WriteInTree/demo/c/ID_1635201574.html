Demonstration and Tutorial of WriteInTree → Cool features → Css →

The generated html files refer to "styles.css".

You also may specify html-css classes for a tree-node, by attaching a "# class" child-node to it, which in turn contain the classes as plain text child nodes. You can attach any number of class nodes to a node, and a class node may contain any number of classes.

Page titles have class "page_title".
