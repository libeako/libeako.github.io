Demonstration and Tutorial of WriteInTree → Tutorial → Embedding files →

Embedding files, like images into a site is quite common need. For it I recommend putting your files to be embedded into a "file" sub-folder inside the folder where the source mm file is and extend your compile script with an rsync command to copy this folder into your output folder. An example compile script. Then, from the mindmap file you will refer to it as "file/whatever.ext".
