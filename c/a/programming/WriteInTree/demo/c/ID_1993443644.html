A tree node may be either a plain text node or a meta node.

Text nodes are usually the ones that will be visible be the reader and are the main content of your work. Meta nodes are instructions for the compiler how to format your content into html website. Meta nodes have the format "# instruction".

Text nodes are entered without any special syntax, just as they are. Meta nodes are prefixed by "# ". If you need to create a text node with prefix "# ", then do it with prefix "## # ", as prefix "## " instructs the software to treat the following as a plain text node.
