conceptual - how we humans think about the program

technical - what is done automatically by the compiler and the run-time environment instead of the human coder

imperative

functional

logic

technical level

automation

memory management

in this section i clear some terminology that i feel are often used incorrectly

example of a language that does not have any automatism in memory management is c

garbage disposal

"disposal" is a better word than "collection"

collection is just a part of disposal, but we also want to reclaim the memory of the identified garbage

some garbage disposal systems are not stop-the-world, do not have a collecting phase

for example : reference counting

c++ programming has partially automatic garbage disposal by RAII, smart pointers circular references are not discoverable with those though

java, haskell has fully automatic garbage disposal, including of circles

layout management

whether an object contains an other [as in memory] or refers to it with a pointer

evaluation order

maintaining thread pools, the programmer only needs to specify "task"s

effect on runtime efficiency

low level control in a programming language is for manual optimization

but that does not mean that low level is good for efficiency rather : it only changes the efficiency story [optimizability =/= efficiency]

human capacity is limited to handle complexity though the human is more clever than the compiler algorithm in optimizing program for simple task, with the increase of the complexity of the task the human loses his|her ability to optimize, while the compiler is not affected by complexity hence a level of complexity exists at which the human is already worse than the compiler

low level coding is more time consuming, and in exchange provides more ability for low level optimization high level programming generally saves programming time, which can be spent on high level optimization [asymptotic improvements] high level optimization may earn more than low level, again : usually more so as complexity grows
