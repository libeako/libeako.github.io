abstraction

foundational set theory is good in implementing interfaces but bad in defining interfaces

example : in the representation of natural numbers by pure sets

one natural number is an element of its successor

this is a property not of the natural numbers but of this representation of it

so this representation captures the natural numbers imprecisely

because of this : set theory does not help mathematicians and software coders in defining abstractions

in this role category theory is emerging
